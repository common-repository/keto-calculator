Keto Calculator

== Description ==
Keto Calculator, developed by [The Right Software](https://therightsw.com/) web development company, is a simple keto diet calculator, which is used to measure your nutritional needs on a ketogenic diet. It helps users achieve their goal of healing, weight loss, performance and more. Use shortcode [keto_calculator] to show plugin on frontend.

For more info about WordPress plugin development, visit The Right Software [plugin development services](https://therightsw.com/plugin-development/) page.
View other [WordPress Plugins](https://profiles.wordpress.org/therightsw/#content-plugins).
Send your problems to [hello@therightsw.com](mailto:hello@therightsw.com). Dont forget to mention WordPress and WooCommerce versions.

*Features:*

1. Users can enter their information in either Metric or US Customary units.
2. Calculate results in tabular as well as graphical form.
3. Provide users both weight loss and weight gain diet plan.
4. Can be placed on any wordpress page with [keto_calculator] shortcode.

*Future Updates:*

1. Options for customizing table and graph styles
2. Options for customizing result text
3. Generate PDF reports


== Installation ==
1. Upload (ketocalculator)folder to the `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the following short code where you want to display ketocalculator
   shortcode: [keto_calculator]
